<h1> MineSocial </h1>
Un petit réseau social étudiant inspiré des foncitonnalités de Facebook
<hr/>
Pour réaliser cela j'utilise les outils suivants
<ul>
    <li>Python & Django</li>
    <li>Html, Css</li>
    <li>Javascript</li>
    <li>Pencil pour déssiner les maquettes</li>
    <li>Umlet pour le diagramme UML</li>
</ul>
