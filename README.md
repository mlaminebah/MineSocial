# MineSocial
Un petit réseau social étudiant inspiré des foncitonnalités de Facebook
---
Pour réaliser cela j'utilise les outils suivants
* Python & Django
* Html, Css
* avascript
* Pencil pour déssiner les maquettes
* Umlet pour le diagramme UML
